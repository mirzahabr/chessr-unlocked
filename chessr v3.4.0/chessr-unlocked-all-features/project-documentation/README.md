# project-documentation/

Documentation only — never bundled/loaded by the browser, deliberately separate from the
extension's own source tree.

- **`PROJECT_REPORT.md`** — the authoritative technical reference for this version: directory
  structure, manifest wiring, data flow, per-component detail, the engine suite, the
  CDP/debugger fallback, deobfuscation history + What's New vs. the previous version, and
  (in this build only) the full account-system architecture/removal writeup.
- **`knowledge-base/`** — a frozen snapshot of the cross-version canonical knowledge base at
  the time this version was processed. The live, continuously-updated copy lives at the
  repository root's `knowledge-base/` folder, one level above every version folder.
